# Market Empire v0.3 — версия с интерактивной картой

Это новая версия игры. Она отличается от старого MVP главным экраном-картой города, картами районов, ПВЗ, складами, конкурентами и борьбой за долю рынка.

## Как запустить без открытия старой версии

1. Полностью распакуйте архив в **новую папку**. Не копируйте файлы поверх старого MVP.
2. Запустите `start_local.bat`.
3. Скрипт сам выберет свободный порт, начиная с `8017`, отключит кэш и откроет нужную страницу.
4. В правом нижнем углу игры должна быть надпись:

```text
v0.3 map build 2026.06.17
```

Если этой надписи нет, открыта не эта сборка.

## Почему раньше открывалась старая версия

Скорее всего, старый сервер `python -m http.server 8000` продолжал работать. При открытии `http://localhost:8000` браузер обращался именно к старой папке. Эта сборка больше не использует фиксированный порт 8000.

## Ручной запуск

```powershell
python local_server.py
```

Не открывайте старую ссылку `http://localhost:8000`. Используйте адрес, который появится в окне сервера.

## Что есть в сборке

- создание названия и цвета магазина;
- интерактивная карта города из шести районов;
- отдельные карты районов;
- ПВЗ, свободные площадки и склады;
- конкуренты и доли рынка;
- открытие и улучшение ПВЗ;
- товары, закупки, поставки и цены;
- локальная реклама;
- захват и потеря районов;
- ежедневные финансовые отчёты;
- сохранение прогресса в `localStorage`.

## Telegram

Для запуска внутри Telegram разместите содержимое папки на HTTPS-хостинге и задайте URL в `.env` для `bot.py`.


## Если BAT-файл не запускается

Используйте `start_local.cmd`. Оба файла сохранены в формате Windows CRLF и содержат только ASCII-команды.

Также можно открыть PowerShell в папке проекта и выполнить:

```powershell
python local_server.py
```


## Исправление v0.3.2

В предыдущем архиве новый `app.js` был ошибочно объединён со старым `index.html`, из-за чего JavaScript останавливался до подключения обработчиков кнопок. В этой сборке HTML, стили и JavaScript синхронизированы.
