# Marketplace Empire

Marketplace Empire — MVP Telegram Mini App.

Рабочий MVP симулятора продавца на маркетплейсе.

## Что уже есть

- стартовый баланс 50 000 ₽;
- создание названия магазина и выбор цвета сети;
- один город с шестью районами;
- интерактивная карта города с долями рынка и лидерами районов;
- отдельная карта района с 5-10 ПВЗ или свободными помещениями;
- стартовый склад и первый ПВЗ в Южном районе;
- открытие и улучшение ПВЗ;
- локальная реклама района;
- локальное влияние, лояльность и плавное изменение доли рынка;
- три конкурента: LowPrice, PremiumBox и FastGo;
- конкуренты тратят ограниченный бюджет на рекламу, точки и защиту районов;
- захват, потеря и бонусы районов;
- открытие соседних районов после контроля территории;
- городской склад на поздней городской стадии;
- каталог из 10 товаров;
- закупка партий;
- изменение цены продажи;
- платное продвижение;
- симуляция продаж по игровым дням;
- комиссия маркетплейса, логистика и возвраты;
- рейтинг магазина, отчёты и статистика;
- события рынка на каждый игровой день;
- сезонные цели с денежными наградами;
- прокачка бренда, логистики, маркетинга и аналитики;
- выбор фокуса дня: аналитика цен, карточки товара, сервис или поставщики;
- товар дня с отдельным спросом и расходом на витрину;
- прогноз спроса по цене, включая 0% при сильном завышении цены;
- динамическая цена рынка, тренды, конкуренты и история рынка;
- несколько поставщиков с разной ценой, сроком, качеством и риском;
- очередь поставок, вместимость склада и расходы на хранение;
- рекламные стратегии вместо простого включения рекламы;
- улучшения карточек товаров;
- случайные события с выбором реакции;
- версия сохранения и миграция старых localStorage-сейвов;
- отдельные экраны: склад, аналитика, развитие и задания;
- сотрудники с зарплатой и игровыми эффектами;
- автоматизация цены, автозаказа и остановки невыгодной рекламы;
- сохранение прогресса в localStorage;
- адаптация под тему Telegram;
- запуск как обычный сайт или Telegram Mini App.

## Быстрый запуск на Windows

1. Распакуйте папку.
2. Дважды нажмите `start_local.bat`.
3. Откройте в браузере `http://localhost:8000`.

Можно также выполнить вручную:

```powershell
python -m http.server 8000
```

## Запуск через Telegram

Telegram открывает Mini Apps только по публичному HTTPS-адресу.

1. Разместите файлы `index.html`, `styles.css`, `city.js` и `app.js` на GitHub Pages, Cloudflare Pages, Netlify, Vercel или другом HTTPS-хостинге.
2. Создайте бота через `@BotFather`.
3. Скопируйте `.env.example` в `.env` и заполните:

```env
BOT_TOKEN=токен_бота
WEB_APP_URL=https://адрес-вашего-сайта/
```

4. Создайте виртуальное окружение и установите зависимости:

```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

5. Запустите бота:

```powershell
python bot.py
```

После `/start` бот покажет кнопку запуска игры. Также будет установлена кнопка меню «Играть».

## Важно для следующей версии

Сейчас прогресс хранится только на устройстве пользователя. Для рейтинга игроков, синхронизации между устройствами и защиты от изменения баланса через DevTools понадобится backend с проверкой Telegram `initData` и база данных.
