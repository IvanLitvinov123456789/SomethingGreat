# Market Empire v0.3.3 — mobile map build

Версия адаптирована для телефонов и Telegram Web App.

## Изменения

- карта на телефоне стала прокручиваемой и автоматически центрируется на выбранном районе;
- подписи, точки ПВЗ и области нажатия увеличены;
- верхняя панель, карточки, модальные окна и нижнее меню адаптированы под узкие экраны;
- районы получили более сложные многоугольные границы с большим числом углов;
- сохранена работа в обычном браузере и Telegram Web App;
- старое сохранение v0.3 продолжает использоваться.

## Запуск

Запустите `start_local.bat` или `start_local.cmd`.

Для запуска вручную:

```powershell
py -3 local_server.py
```

Для проверки на телефоне компьютер и телефон должны находиться в одной Wi-Fi сети, а сервер должен слушать внешний интерфейс. В текущем локальном скрипте адрес для телефона выводится в консоли, если он доступен.
